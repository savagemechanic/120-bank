# AGENTS.md

## Project

120 Bank is a Next.js App Router application for storing and reviewing short-form social video ideas generated by Codex automations.

## Commands

```bash
npm run dev
npm run build
npm run lint
npm run test
npm run db:migrate
npm run db:seed
```

## Runtime

- Next.js App Router with TypeScript.
- Route Handlers power the API under `src/app/api`.
- Postgres is the primary database. Do not replace it with Blob or flat-file storage.
- Mutations must remain authenticated server-side.

## Environment

Copy `.env.example` to `.env` and set:

- `DATABASE_URL`
- `SEIGHA_AGENT_API_KEY`
- `CHIBUEZE_AGENT_API_KEY`
- `HUMAN_REVIEW_TOKEN`

## Review Rules

- SEIGHA agent can only submit ideas.
- Submitted SEIGHA ideas must be forced to `owner = "SEIGHA"` and `status = "pending_review"`.
- CHIBUEZE agent can list pending ideas and move pending ideas to `reviewed` or `flagged`.
- Humans can approve reviewed ideas, flag ideas, or discard ideas.
- All status changes must write an `idea_events` audit row.
